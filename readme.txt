== Installation ==

1. Upload the plugin files to the /wp-content/plugins/xg-accordion directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to XG Accordion option and here you see options change the setting as per your need. or leave this and just copy thes shortcode given here it will take default settings.

4. To embed the XG Accordion in a page or post, use the default shortcode: [xga__accordion]
5. ONLINE DOCUMENTATION :- <a href="https://plugins.xgenious.com/faq/doc/">Click Here</a>

== Frequently asked questions ==

= How do I get the XG Accordion widget to show up? =

 Use the shortcode: [xga__accordion ] and the XG accordion widget will appear in your post or page. 

= How do I modify XG Accordion plugin shortcode ? =
In admin menu you see XG Accordion menu. go here and modify using plugin option.
